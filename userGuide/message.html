Messages Helper

Messages Helper functions are used to display messages.



Loading Messages Helper

Within your controller: 

$this->load->systemHelper('messages');

will load the helper functions.

Using Messages Helper Functions

  • setFlashMessage($message)

    • Displays a "Flash" message when the page gets loaded.

  • setFlashMessage($message)

    • Displays a "Flash" message when the next page gets loaded.
