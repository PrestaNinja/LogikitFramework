Creating Your Own Helpers

The path for your own helpers is application/helper
To use a custom helper file:
  • Copy the file under application/helper folder
  • Load it.
If the filename of your helper is myMagicImages.php, copy it under application/helper folder and load it as: 
$this->load->helper('myMagicImages');
once loaded, you will use the function within your helper as you would do with any other function: 
useMagic(); //a function from myMagicImages.php
