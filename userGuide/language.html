Using the Language Class

Language class is loaded upon your settings in application/config/config.php file. You will find the instructions there.


Changing the Active Language

  • setLanguage() Method

$this->Language->setLanguage('en');
will set the active language as 'en' and will load $lang array found in your language files, as a propoerty of the controller.
Language files are located in application/language/{language initials}/directories.
$this->lang array is accesible within your view files.
