Loading the FTP Class

$this->load->systemLibrary('Ftp');
will load the class and the ftp object will be available within your code.

This is an experimental class introduced with LF 1.0.1 and may be subject to change with future releases.



A Quick Example

Within your controller: 

        $this->load->systemLibrary('Ftp');
        $this->Ftp->connect('myhost.com' , 'myuser' , 'mypass');
        $this->Ftp->mirrorUp('/var/www/test' , '/public_html');
        $this->Ftp->close();
uploads everything recursively, under local /var/www/test folder to /public_html folder of the remote host (myhost.com in our example). If remote respective folders do not exist, the library will attempt to create them.



Function Reference

  • $this->Ftp->settings()
    Alter the connection settings: 
    $settings = array();
$settings['user'] = 'myuser';
$settings['password'] = 'mypassword';
$settings['host'] = 'myhost';
$settings['port'] = 2525;
$this->Ftp->settings($settings);
$this->Ftp->connect();
  • $this->Ftp->setPort()
    Sets the port for a connection: 
    $this->Ftp->setPort(2532);
    Port setting defaults to 21 (default FTP port).
  • $this->Ftp->connect()
    Establish an ftp connection: 
    $this->Ftp->connect('myhost' , 'username' , 'pass');
    This method can be used both combined with settings() function (in this case it is used without parameters) or by itself, passing the parameters directly to the function.
  • $this->Ftp->listFiles()
    Returns the list of files in a given folder, as an array: 
    $files = $this->Ftp->listFiles('/public_html');
  • $this->Ftp->move()
    Moves or renames a remote file: 
    $files = $this->Ftp->move('/public_html/oldFile.html' , '/public_html/newFile.html');
  • $this->Ftp->deleteFile()
    Deletes a remote file: 
    $files = $this->Ftp->deleteFile('/public_html/file.html');
  • $this->Ftp->deleteDirectory()
    Deletes all files in a directory recursively: 
    $files = $this->Ftp->deleteDirectory('/public_html');
    Pay extreme attention using this function as it will remove everything under given folder, including the sub-folders and their contents.
  • $this->Ftp->download()
    Downloads a file from the remote host and saves it with the given name. First parameter is local path: 
    $files = $this->Ftp->download('/var/www/downloaded.html' , '/public_html/file.html');
  • $this->Ftp->upload()
    Uploads a file from the local host. First parameter is local path: 
    $files = $this->Ftp->uploaded('/var/www/file.html' , '/public_html/uploaded.html');
