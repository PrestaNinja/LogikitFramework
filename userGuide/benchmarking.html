Loading the Benchmarking Class

$this->load->systemLibrary('Benchmarking');
will load the class.

Using the Benchmarking Class

  • 1. Mark a start point
  • 2. Mark an end point
  • 3. Run the "elapsed time" function to view the results
Here's an example using real code: 
$this->Benchmark->addMark('codeStart');

// Some code happens here

$this->Benchmark->addMark('codeEnd');

echo $this->Benchmark->elapsed('codeStart', 'codeEnd');
