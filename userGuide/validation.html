Validation Class

Using LF, you are encouraged to keep the form validation logic seperated.

LF will look for the validatiors in application/validator/{controller name}/ folder.



Loading the Validator Class

If your controller is Entries class in application/controller/Entries.php file: 

		$this->load->validator('EditFormValidator');
will load the file called application/validator/Entries/EditFormValidator.php and will load EditFormValidator class.



Anatomy of the Validator Class

A typical validator class will look like: 

class EditFormValidator extends LogikitValidation
{
    public function __construct()
    {
        
    }
}
Any Validator Class must extend LogikitValidation



Using the Validator Class

Supposing you have some POST data, in our controller: 

	public function validate()
	{
		$this->load->form('EditForm');
		$this->load->validator('EditFormValidator');
		
		$validationResult = $this->EditFormValidator->validateForm($this->EditForm);
		
		if($validationResult['validationOverAll'] == TRUE)
		{
			//form is validated
		}
		else
		{
			//form is unvalidated. see what is wrong.
			print_r(validationResult);
		}
	}
