Using LF Libraries

All of the available libraries are located in your system/library folder. In most cases, to use one of these classes involves initializing it within a controller using the following initialization function: 

$this->load->systemLibrary('className');
Where className is the name of the class you want to invoke. For example, to load the Net Client class you would do this: 
$this->load->systemLibrary('NetClient');
once loaded, you will use it within your controller as: 
$this->NetClient->getUrl('someUrl');
