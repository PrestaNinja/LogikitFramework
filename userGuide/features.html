Logikit::Framework Features

The list you will find below is here just to keep the integrity of the user guide. It will not tell you how all those features will get your programming life easier.

The best way to see it for yourself is to install LF and give it a try. It is really easy that if your grandmother knows how to use a computer, then she could do it. You already did it? Then you can safely continue to the next page.

However, here is the list just in case:

  • Model-View-Controller Based System
  • Highly Light Weight
  • Object Oriented PHP 5 coding
  • A Simplified ORM
  • Built-in Ajax Support
  • Form Generation
  • Code Generation
  • Powerful Form and Data Validation
  • Security
  • File Uploading Class
  • Localization
  • Pagination
  • Benchmarking
  • Search-engine Friendly URLs
  • Flexible URI Routing
  • Support for Hooks, Class Extensions, and Plugins
  • Large library of "helper" functions
