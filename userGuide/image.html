Image Manipulation Class

Using image manipulation class, you can do the following actions:

  • Image Resizing
  • Thumbnail Creation
  • Image Cropping
  • Image Rotating
  • Image Watermarking



Loading the Image Class

$this->load->systemLibrary('Image');
will load the class and the image object will be available within your code.



Loading the Image

Within your controller: 

        $this->load->systemLibrary('Image');
        $this->Image->loadImage('/var/www/test/EdgarPoe.jpg');
will load the image (EdgarPoe.jpg in our example) to be manipulated.



Saving the Output

Within your controller: 

	$this->Image->save('/var/www/test/EdgarPoeThumb.jpg');
will save the manipulated image.



Thumbnail Creation

$this->Image->resize(100 , 50);
will create a thumbnail with width = 100px and height = 50px
$this->Image->resize(100 , NULL);
will create a thumbnail with width = 100px and height proportional to the width of the original image. If our original width was 200 x 80, the new height will be 40. 
$this->Image->resize(NULL , 100);
will create a thumbnail width height = 100px and width proportional to the height of the original image. If our original height was 80 x 200, the new width will be 40.



Image Cropping

$this->Image->crop(40 , 50);
will crop the image 40 pixels from X axis and 50 pixels from Y axis.



Image Rotating

$this->Image->rotate(90);
will rotate the image 90 degrees counter-clockwise.



Image Watermarking

Be our image to be watermarked /var/www/testImage.jpg and our watermark image /var/www/watermark.png 

$this->Image->setWatermarkImage('/var/www/watermark.png');
$this->Image->watermark();
will merge our watermark image at the bottom-right corner of the image to be watermarked, /var/www/testImage.jpg.



Complete Image Manipulation Example

We want to open /var/www/testImage.jpg , watermark it using /var/www/watermark.png and save it as /var/www/testWatermark.jpg. Our complete cotroller function would be: 

    public function watermark()
    {
        $this->load->systemLibrary('Image');
        $this->Image->loadImage('/var/www/testImage.jpg');
        $this->Image->setWatermarkImage('/var/www/watermark.png');
        $this->Image->watermark();
        $this->Image->save('/var/www/testWatermark.jpg');
    }
