Handling Namespaces

Using separate namespaces for each event or item is a convenient way to keep the session values proper. Namespace functions are automatically loaded when LF is initialized.



Function Reference

  • setNamespace()
    Initializes a namespace. 
    		setNamespace('UserData');
  • unsetNamespace()
    Unsets a namespace. 
    		unsetNamespace('UserData');
  • getNamespaceValues()
    Returns the values of the given namespace in an array. 
    		getNamespaceValues('UserData');
  • getNamespaceValue()
    Returns the value of the given namespace, if only a single value is defined in and the function is used without parameters. If a key (first and only parameter) is given, it will return the value corresponding to the key. 
    		getNamespaceValue('UserData');

    		getNamespaceValue('UserData' , 'email');
  • addToNamespace()
    Adds a value to a namespace. If third parameter is not given, the value (second parameter) will take the next key. 
    		addToNamespace('UserData' , 'my@myexample.com' , 'email');

    		addToNamespace('emailList' , 'my@myexample.com');
