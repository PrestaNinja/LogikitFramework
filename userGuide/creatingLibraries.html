Creating Your Own Libraries

LF is very flexible that you could import any existing class into the system.

The path for your own libraries is application/library
To do that:
  • Copy the class under application/library folder
  • Arrange the name of the file and the class name so that they will be the same and will start with a capital letter.
Example:
Let's assume you have a class called "tester_class" in a file called tester.class.php.
  • Copy the file under application/library folder and rename it to Tester.php
  • Open the file with your text editor and rename the class to Tester
Now you are ready to import the class: 
$this->load->library('Tester');
once loaded, you will use it within your controller as: 
$this->Tester->testThis('someParam');
