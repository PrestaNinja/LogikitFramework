Email Sending Class

LF Email sending class provides an easy way to compose emails with multiple attachments and multiple recipients.

This is an experimental class introduced with LF 1.0.1 and may be subject to change with future releases.



Loading the Email Class

$this->load->systemLibrary('Email');
will load the class and the email object will be available within your code.



A Quick Example

Within your controller: 

$this->load->systemLibrary('Email');
$this->Email->from('me@myserver.com');
$this->Email->to('recipient@remoteserver.com');
$this->Email->subject('Test mail');
$this->Email->message('This is my test message');
$this->Email->send();
will send an email to recipient@remoteserver.com from me@myserver.com with the subject Test mail and with the content This is my test message.



Function Reference

  • $this->Email->from()
    Sets the email address and name of the person sending the email: 
    		$this->Email->from('me@mymail.com');
  • $this->Email->replyTo()
    Sets the reply-to address. If the information is not provided the information in the "from" function is used. Example: 
    		$this->Email->replyTo('you@yourmail.com');
  • $this->Email->to()
    Sets the email address(s) of the recipient(s). Can be a single email, or an array: 
    		$this->Email->to('you@yourmail.com');

    $list = array('one@yourmail.com' , 'two@yourmail.com' , 'three@yourmail.com');
$this->Email->to($list);
  • $this->Email->cc()
    Sets the CC email address(s). Just like the "to", can be a single email or an array.
  • $this->Email->bcc()
    Sets the BCC email address(s). Just like the "to", can be a single email or an array.
  • $this->Email->subject()
    Sets the email subject. 
    $this->Email->subject('This is my subject');
  • $this->Email->message()
    Sets the email message body: 
    $this->Email->message('This is my message');
  • $this->Email->send()
    Email sending function: 
    $this->Email->send();
    sends your email.
  • $this->Email->attach()
    Enables you to send an attachment. Put the file path/name in the first parameter.
    Use a file path, not a URL. For multiple attachments use the function multiple times.
    For example: 
    $this->Email->attach('/var/www/test/test.jpg');
$this->Email->attach('/var/www/test/test.odt');
$this->Email->attach('/var/www/test/test.pdf');
